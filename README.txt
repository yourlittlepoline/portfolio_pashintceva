ПОРТФОЛИО ПОЛИНЫ ПАШИНЦЕВОЙ

Как открыть:
1. Откройте файл index.html в браузере.
2. Для публикации загрузите всю папку на GitHub Pages / Netlify / любой статический хостинг.

Структура:
- index.html — контент
- styles.css — дизайн и адаптивность
- script.js — навигация, вкладки, увеличение фото
- assets/images — изображения
- assets/docs — резюме и документы
